customFolderName:packageContents:

When a package is installed the root object of an unzipped package should be a folder named with the standard identifier(--).

- (NSString *)customFolderName:(CCPackage *)package packageContents:(NSArray *)packageContents

Parameters

package

The package for which the unzipped folder name should be determined

packageContens

A list of URLs of unzipped package’s first level directory

Return Value

The folder name of the package

Discussion

The package installer will first try to find a folder with the standard identifier. If that fails you can omplement this method if a package contains a folder named other than the standard identifier. The content of the unzipped package is provided to select a folder name. If finding the standard identifier fails and this method is not implemented the installation will fail-
