Class Methods

effectWithOutlineColor:fillColor:outlineWidth:fieldScale:distanceField:

Initializes a CCEffectDFOutline object with the supplied parameters.

+ (id)effectWithOutlineColor:(CCColor *)outlineColor fillColor:(CCColor *)fillColor outlineWidth:(int)outlineWidth fieldScale:(float)fieldScale distanceField:(CCTexture *)distanceField

Parameters

outlineColor

Color of the outline, a [CCColor blackColor] will result in an opaque black outline.

fillColor

Color of the fillColor, a [CCColor blackColor] will result in an opaque black fillColor.

outlineWidth

pixel width of the outline.

fieldScale,

defined by the distance field generation proccess, for example a distance field that was generated with a 4096/4096 input and output a 128/128 texture would have a fieldScale of 32 ( input size / output size). Note: this parameter could be automatically calculated if we assume that all distance fields are generated from a 4096 input.

Return Value

The CCEffectDFOutline object.
